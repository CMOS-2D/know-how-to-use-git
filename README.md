# know-how-to-use-git
everyone in CMOS+2D can use this repository to try and learn to use git

please use `Git` to get this repository on your own PC. Write down your name in the list and push it to the CMOS+2D group. It is recommended to create a branch and avoid directly pushing changes to the main branch.
